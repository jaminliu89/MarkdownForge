# MarkdownForge 使用说明

**离线运行的 Markdown 精美排版工具 · 25 套主题 · 单页 PDF · 3x 高清长图**

---

## 快速开始

1. 双击 **MarkdownForge.dmg** 挂载
2. 将 **MarkdownForge.app** 拖入 **Applications** 文件夹
3. 从启动台或 Spotlight 打开 MarkdownForge
4. 首次运行如提示"无法验证开发者"，请在**系统设置 → 隐私与安全性**中允许打开

---

## 核心功能

### 双栏实时预览

左栏输入 Markdown，右栏实时精美渲染。预览区可**直接双向编辑**——鼠标点进去改文字，改动会同步回左栏。

### 25 套精心调校的主题

| 分组 | 主题 |
|---|---|
| 基础 | 默认（无衬线）· 书籍（衬线）· 米黄 · 深色 |
| 极简 | 无印良品 |
| 中国风·淡雅 | 宣纸（朱砂）· 青瓷（黛绿）· 竹青（水墨）· 胭脂（浅绯） |
| 平台风格 | 微信公众号 · 小红书 · 掘金 · 知乎 · GitHub · 报纸 · 打字机（等宽） |

**排版铁律**：加粗只加字重不变色；引用/代码块无左侧竖线；列表无前置符号；已完成任务用灰色删除线；分隔线极淡；零 emoji 装饰。

### 一键导出

| 按钮 | 输出 | 说明 |
|---|---|---|
| **复制 HTML** | 独立 HTML 到剪贴板 | 可直接粘贴到任何编辑器；配套终端工具 `md2pdf` 可生成矢量 PDF |
| **复制富文本** | 富文本到剪贴板 | 粘贴到微信公众号 / 小红书 / Word 保留样式 |
| **下载 HTML** | .html 文件 | 独立可滚动，保留完整样式 |
| **导出 PDF** | .pdf 文件 | **单页贴合内容**，不分 A4 页，3x 高清 |
| **生成长图** | .png 文件 | 3x 高清整幅长图，适合微博 / 小红书 |

### 批量转换

点右上角 **批量** 按钮：

- 拖拽多个 `.md` / `.markdown` / `.txt` 文件（支持拖文件夹）
- 一次可选主题 / 格式（HTML / PDF / PNG）
- 打包方式：单文件依次下载 · 或 zip 一个压缩包
- 可选择保存目录（浏览器 File System Access API 授权后直接写盘）

## 常见问题

**Q：为什么导出的 PDF 是单页超长而不是分页 A4？**  
故意的。抛光机默认按内容高度贴合，避免文字被硬切。要 A4 分页请用浏览器 `⌘P` 打印。

**Q：粘贴到微信公众号后代码块样式丢了？**  
微信编辑器会过滤部分 CSS。用**复制富文本**而不是**复制 HTML**，或先粘到「墨滴/Md2All」再复制到公众号。

**Q：长图导出为什么这么慢？**  
3x 高清渲染，长文档十几秒是正常的。如果内容极长可考虑分段生成。

**Q：怎么用命令行生成矢量 PDF？**  
点【复制 HTML】把独立 HTML 存到磁盘，在终端跑 `md2pdf 文件名.html`（配套工具，见 `~/.local/bin/md2pdf`）。矢量单页、字体可选、无损缩放。

**Q：数据会上传到服务器吗？**  
不会。完全本地运行，所有渲染库（marked · highlight.js · html2canvas · jsPDF · jsZip · turndown）都内嵌在应用里，无任何网络请求。
